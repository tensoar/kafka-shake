# kafka-shake

A simple Kafka message viewing client, download the pre-packaged app in [release](https://github.com/tensoar/kafka-shake/releases).

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
